# Analog Frame Lab

Статический сайт для добавления аналоговых рамок по краям фотографий.

## Что уже работает

- загрузка JPG / PNG / WEBP;
- drag-and-drop;
- несколько встроенных рамок;
- регулировка толщины и прозрачности;
- цифровое зерно;
- экспорт в JPG и PNG;
- вся обработка происходит локально в браузере.

## Как запустить локально

Просто откройте `index.html` в браузере.

Для корректной работы внешних PNG-рамок лучше запустить локальный сервер:

```bash
python3 -m http.server 8080
```

Затем открыть:

```text
http://localhost:8080
```

## Публикация на GitHub Pages

1. Создайте новый репозиторий.
2. Загрузите содержимое этой папки в корень репозитория.
3. Откройте `Settings → Pages`.
4. В разделе `Build and deployment` выберите `Deploy from a branch`.
5. Выберите ветку `main` и папку `/root`.
6. Сохраните настройки.

## Как подготовить реальные рамки

Лучший формат:

- PNG с прозрачным центром;
- высокое разрешение, желательно 4000–6000 px по длинной стороне;
- рамка должна занимать весь холст;
- внутренняя область остаётся прозрачной;
- желательно собрать варианты под горизонтальные, вертикальные и квадратные изображения.

Папка `frames/` уже создана. Следующий шаг — подключить PNG-рамки как отдельные пресеты.
