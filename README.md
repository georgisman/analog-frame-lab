# Analog Frame Lab — v3

Статический GitHub Pages-сайт для добавления рамок и подложек к фотографиям.

## Новое в v3

- чистые рамки в стиле editorial / gallery print;
- белая, тёплая белая, чёрная или произвольная подложка;
- уменьшение фотографии внутри выбранного формата;
- форматы Original, 1:1, 4:5, 5:4, 3:4, 4:3, 2:3, 3:2, 9:16 и 16:9;
- режимы «Вписать целиком» и «Заполнить формат»;
- сдвиг фотографии по горизонтали и вертикали;
- экспорт JPG и PNG;
- обработка полностью локальная.

## Обновление GitHub Pages

Замените в корне репозитория три файла:

- `index.html`
- `styles.css`
- `app.js`

GitHub Pages обновится автоматически после коммита в ветку `main`.
